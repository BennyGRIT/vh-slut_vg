# Gritty Café

## Om Gritty Café
Gritty Café är en Site för att lära sig versionshantering med Git och Github som host.
Dessutom bakar vi goda kakor som vi bjuder på lärarna.
Lärarna blir då extra glada och vill lära ut ännu mer kunskap.
